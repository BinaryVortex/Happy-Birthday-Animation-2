# Happy Birthday Animation 2

A small, lightweight Happy Birthday animation built with HTML, CSS and JavaScript. This repository contains a self-contained demo you can open in any modern browser.

## Overview

This project shows a festive animated Happy Birthday greeting using CSS animations and a tiny bit of JavaScript for interactivity. It's intended as a simple demo you can customize, embed in a webpage, or use as a standalone animation.

## Demo / Screenshot

![Happy Birthday animation screenshot](./Screenshot%202024-07-03%20083429.png)

The screenshot above shows the animation as it appears in a browser. To run the demo locally, open `index.html` in your browser (no build tools required).

## Features

- Pure front-end: HTML, CSS and JavaScript only
- Responsive animation that works on desktop and mobile
- Easy to customize colors, text, and animation timing

## Built with

- HTML
- CSS (majority of the code)
- JavaScript

## Getting started

1. Clone the repository:

   git clone https://github.com/BinaryVortex/Happy-Birthday-Animation-2.git

2. Open `index.html` in your browser:

   - Double-click the file, or
   - Right-click -> Open with -> your preferred browser

No dependencies or build steps required.

## Project structure

- `index.html` — main demo page
- `style.css` — styles and animations
- `script.js` — small script for interactivity
- `Screenshot 2024-07-03 083429.png` — demo screenshot used in this README

## Customization

- To change the message, edit the HTML in `index.html`.
- To change timings or animation details, update `style.css`.
- Small interactive behaviors live in `script.js`.
